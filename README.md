# 🌡️ Temperature Converter

A simple and responsive **Temperature Converter** web application built using **HTML, CSS, and JavaScript**. It allows users to instantly convert temperatures between **Celsius**, **Fahrenheit**, and **Kelvin** through an intuitive and user-friendly interface.


## 📌 Features

- Convert temperature between:
  - Celsius (°C)
  - Fahrenheit (°F)
  - Kelvin (K)
- Real-time conversion
- Simple and responsive user interface
- Easy to use
- Input validation for accurate results

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript 


## 📂 Project Structure

Temperature_Converter/
│── Index.html



## 💡 How It Works

1. Enter the temperature value.
2. Select the input temperature unit.
3. Choose the unit you want to convert to.
4. Click the **Convert** button.
5. The converted temperature is displayed instantly.
